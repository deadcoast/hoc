# ASCII Concept Drafts 0.0.3

- USES PSEUDO CODE AND NAMING TYPES TO ENSURE SPACING CONSISTENCY

`CHOICE` = `CHO`
`ACTION` = `ATN`
`DECLARE` = `DEC`
`DEFINE` = `DEF`
`ABSOLUTE` = `ABT`
`essential` = `ent`
  `emotional` = `emo`
  `physical` = `phl`
`variable example` = `var`
`variable example two` = `vb2`

```css
          ┌── [CHO] ──┐
          │     ↓     │
          │   [ACT]   │
          │     ↓     │
          │   [DEC]   │
          │   ⇑ ↓ ⇑   │  
┌─(emo) ← ┤   ⇑ ↓ ⇑   ├ → (phl)─┐
│         │   ⇑ ↓ ⇑   │         │
│ {vrb  ↔ ┤  [{ABT}]  ├ ↔  vrb} │
│         │     ⇓     │         │
│ {vb2  → ┤    ↓⇓↓    ├ ←  vb2} │
└─────────┤    ↓⇓↓    ├─────────┘
          └─ <!DEF!> ─┘
```

```css
                   [CHOICE]
                       ↓
                   [ACTION]
                       ↓
                   [DECLARE]
                   ↓   ↓   ↓
          (emotional   ↓    physical)
         ↓       ↑↑    ↓    ↑↑      ↓
{selfworth    ↔  [({WORKOUT})]   ↔  confidence}
                       ⇓
{placeholder  →       ↓⇓↓        ←  placeholder}
                       ⇓        
                    <!DEF!>
```
